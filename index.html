Specifications

Features
  • Uses Ajax to validate input.
  • Can be easily modified to check a database for on the fly app-specific validation.
  • Basic setup using only [data-] properties in HTML tags.
  • Uses Bootstrap Tooltips for a streamlined & functional user interface.
  • Only one single .js file to add to your page, and .php file to add to your server.
  • Works with bootstrap-chosen improved select boxes.
Dependencies
  • Requires jQuery 2.0.2+
  • Requires Bootstrap Tooltips (Bootstrap 2 or 3)

Examples

Values required [data-vreq=*]

Type anything in this field.
[data-vreq=1]

No value required here.
[data-vreq=0]

Submit
Make anything a submit button.
[data-vsubmit=1]

Valid values required [data-vreq=1 data-vtype=*]

This input requires a value, and requires of valid type.
[data-vreq=1 data-vtype='phone_CA']

Tests for valid Canadian/North American Phone Number, try any of these:

  • +1(403) 999 9999
  • 18005555555
  • 1-403-999-9999
  • 403-999-9999
  • 780.999-9999
  • and any other way you can think of.


Value required, and must be valid email.
[data-vreq=1 data-vtype='email']

Optional (but valid) values [data-vreq=0 data-vtype=*]

This input does not require a value, but if entered it must be valid.
[data-vreq=0 data-vtype='phone_CA']

This input does not require a value, but if entered must be valid.
[data-vreq=0 data-vtype='postal_CA']

Tests for valid Canadian Postal Code, try any of these:

  • t4t4t4
  • T1W 4T9
  • t4t-4t4
  • V1Y.1W3
  • and any other way you can think of.
